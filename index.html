Introducción

Para poder realizar este estudio, es indispensable descargar el conjunto de datos a través de la información publicada por la DGT. El objetivo principal de la visualización de Datos es el de comunicar una idea o el resultado de un análisis, estudio, para ayudar a la toma decisiones con el objetivo reducirlos. Es necesario realizar el proceso KDD para identificar patrones válidos, novedosos y potencionalmente útiles. El fichero se encuentra en formato xlsx, el cual se dispone 72 columnas (caracteristicas) y de 72959 filas (instancias), cada una corresponde un accidente con victimas, detallandose para cada fichero de registros y las variables que los componen. Además, asimismo se pretende realizar el análisis de datos exploratorio para comprender los datos mediante las visualizaciones que ayuden a entender lo que se puede averiguar de ellos.

Como se tiene un conjunto de datos de gran volumen, se pretender realizar el estudio de los principales indicadores asociados a los accidentes de tráfico con víctimas, es decir encontrar el impacto de cuantos victimas sufrieron, en que lugar y a que hora ocurro accidentes de tráfico, que tipo de accidentes y número de fallecidos.

Motivación

Mi motivación para el trabajo práctico es profundizar este estudio del trabajo Fin de Master (TFM) que tiene el objetivo a obtener la visualización de datos por concretos, que sean óptimos para la toma de decisiones (o para poder inferir factores determinantes). Posterior a ello se espera realizar la interpretación, y aportar con las conclusiones. Realizar este trabajo y culminar cada etapa de este, me permitirá tener las habilidades y conocimiento del comportamiento de este tipo de escenarios para futuros proyectos.

Para ello, se pretende realizar una investigación bibliográfica en donde se utilizan publicaciones relacionados con la problematica a tratar. Asi tener la oportunidad de implementarlo en la ciudad donde resido, para imponer nuevas normas con el objetivo reducirlos.

Objetivos

El objetivo principal es el de obtener el resultado de impacto de un análisis, estudio, para ayudar a la toma decisiones con el objetivo reducirlos, basado en los principales indicadores asociados a los accidentes de tráfico con víctimas y asi los conductores, por lo cual este análisis pretende responder a las siguientes preguntas:
  • ¿En que mes hubo mayor numero de fallecidos registrados en en accidentes de tráfico?
  • ¿Cual tipo de accidente ocurrido hubo mayor accidentes de tráfico?
  • ¿En cual zona hubo mayor impacto en el accidente de tráfico?
